Individual Project Site Plan

Site Name:

In The Kitchen With Sethu

  • The website will contain information about the catering business, and engage with the audience by demonstrating what the business offers. This website will include a Home, About and Contact pages.

Site Purpose:

  • To introduce and promote a catering business by showcasing services, sample menus, and the chef behind the food.
  • To engage visitors and potential clients by providing clear information and an easy way to get in touch.

Site Goals:

  • Build awareness and credibility for the catering business.
  • Showcase sample menus, event photos, and specialties.
  • Provide background information about the chef and business.
  • Enable easy contact for inquiries or bookings.

Target Audience:

  • Individuals, families, or businesses seeking catering services.
  • People interested in learning about menu options and event capabilities.
  • Clients looking for a reliable, personable chef for their events.

Content Overview:

  • Home: Eye-catching introduction, highlights of services, and a call to action.
  • About: Story of the business and chef, philosophy, experience, and photos.
  • Contact: Easy-to-use contact form, email, phone, and social media links.

Scenarios:

  • 1. A visitor wants to know what services are offered.

    The Home page provides an overview of catering services, sample menus, and event types catered.

  • 2. A client wants to learn about the chef and the business background.

    The About page shares the chef's story, business values, and experience in the catering industry.

  • 3. A potential customer wants to book or inquire about catering.

    The Contact page offers a simple form, phone, and email information for inquiries and booking.

Color Scheme:

The color scheme for this site includes two main colors:

  • Caribbean Current (hsla(182, 100%, 23%, 1)): Used for headings and accents, giving the site a vibrant, fresh, and appetizing look.
  • Floral White (hsla(47, 43%, 96%, 1)): Used as the background color to provide a clean and welcoming feel.

Example CSS HSL:

    /* CSS HSL */
    :root {
      --black: hsla(0, 0%, 0%, 1);                   /* #000000 - used in overlays, text */
      --dark-blue: hsla(197, 68%, 9%, 1);            /* #08080a (roughly) for sticky header background */
      --footer-dark: hsla(194, 62%, 9%, 1);          /* #061D25 for footer backgrounds */
      --footer-accent: hsla(84, 2%, 44%, 1);         /* #70726F for footer borders/text */
      --primary-red: hsla(0, 100%, 50%, 1);          /* #ff0101 - main accent/button color */
      --primary-red-alt: hsla(350, 100%, 57%, 1);    /* #ff0157 - logo accent, hover links */
      --white: hsla(0, 0%, 100%, 1);                 /* #fff - backgrounds, text */
      --light-bg: hsla(47, 43%, 96%, 1);             /* #faf8ef - fallback background */
      --text-main: hsla(0, 0%, 7%, 1);               /* #111 - main dark text */
      --input-border: hsla(0, 0%, 33%, 1);           /* #555 - input border */
      --box-shadow: 0 5px 35px hsla(0, 0%, 0%, 0.8); /* used for menu/expert box shadows */
    }

Typography:

1. Primary Font: Oswald (sans-serif)

  • Usage: Oswald for all headings, providing a bold, modern look.
  • Applied to:
    • Headings (h1, h2, h3, h4): To clearly define sections and help users navigate.

2. Secondary Font: Source Sans Pro (sans-serif)

  • Usage: Source Sans Pro for body text and paragraphs, ensuring readability and clean appearance.
  • Applied to:
    • Body Text (p): All paragraphs and descriptions.
    • Links and Buttons: All interactive elements for design consistency.

Site Structure (Pages):

  • Home: Welcome, business highlights, menu preview, call-to-action.
  • About: Business story, chef bio, philosophy, photos.
  • Contact: Contact form, phone, email, and social media links.
